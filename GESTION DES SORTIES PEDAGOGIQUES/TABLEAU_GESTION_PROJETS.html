🚀 Bienvenue dans le Tableau de Gestion des Projets

Cette application vous permet de gérer l'ensemble des projets pédagogiques, sorties scolaires et voyages de votre établissement.

Première utilisation — 3 étapes essentielles

1
Configurer l'établissement
Cliquez sur ⚙ Paramètres en haut à droite pour renseigner le nom de l'établissement, la signature du chef d'établissement, les niveaux de classe, les enveloppes budgétaires et les référents.
2
Créer votre premier projet
Cliquez sur + Nouveau pour ouvrir le formulaire de saisie. Remplissez au minimum le titre, le référent et le type de projet.
3
Sauvegarder vos données
Cliquez régulièrement sur 💾 Export pour télécharger une copie de sauvegarde. Ce fichier vous permettra de restaurer vos données à tout moment.
⚠️ Important : L'application fonctionne entièrement dans votre navigateur. Vos données sont sauvegardées localement. Pensez à exporter régulièrement un fichier de sauvegarde pour ne rien perdre.

Concept général

Chaque projet passe par plusieurs étapes : création → soumission → validation CA → accord proviseur → terminé. Le tableau de bord vous donne une vue d'ensemble en temps réel.

Barre de recherche et filtres

La barre de recherche (🔍) filtre instantanément les projets par titre, professeur référent, destination ou classe. Vous pouvez également utiliser la liste déroulante de niveau pour filtrer par classe.

Le diagramme de Gantt affiche tous les projets répartis sur l'année scolaire. Chaque barre est colorée selon le statut du projet. Cliquez sur une barre pour voir le détail. La ligne rouge indique la date du jour.

📁 Créer et gérer les projets

Créer un nouveau projet

1
Cliquez sur + Nouveau dans la barre de navigation.
2
Remplissez les 4 sections du formulaire : Identification, Logistique, Budget, Validation.
3
Cliquez sur Enregistrer en bas du formulaire pour sauvegarder le projet.

Accompagnateurs

Dans la section Logistique, vous pouvez ajouter des accompagnateurs un par un. Si vous avez configuré une base d'accompagnateurs dans les Paramètres, leurs noms apparaîtront en suggestion automatique lors de la saisie.

🔖 Statuts et workflow de validation

Chaque projet suit un cycle de vie avec 5 statuts possibles. Modifiez le statut directement dans le formulaire de chaque projet (section IV).

⚪ Brouillon
Projet en cours de rédaction, non soumis à validation.
🟠 Soumis
Dossier complet transmis, en attente d'examen par le CA.
🟢 Validé
Accordé par le CA et le chef d'établissement. Projet confirmé.
🔵 Terminé
Projet réalisé et clôturé. Archivé dans les bilans.
🔴 Annulé
Projet annulé. Conservé dans la liste pour traçabilité.

Validation CA et accord du Proviseur

Dans la section IV. Validation du formulaire, cochez les cases Validation CA et Accord Chef d'Établissement et saisissez les dates correspondantes. Ces informations apparaissent dans le tableau principal.

Notes internes

Trois zones de texte vous permettent de laisser des commentaires internes : Note AR (accusé de réception), Note Accord (conditions de l'autorisation), et Note Gestion (instructions pour l'intendance).

💶 Gestion du budget

Structure budgétaire

Chaque projet dispose d'un budget prévisionnel avec deux colonnes : Dépenses (en rouge) et Recettes/Financement (en vert). Le solde affiché en haut doit tendre vers 0 pour un budget équilibré.

1
Cliquez sur + Ajouter une Dépense pour saisir un poste de dépense (libellé + montant).
2
Cliquez sur + Ajouter une Source pour indiquer les financements (participation familles, subventions, fonds AP/VE…).
3
Si vous avez configuré des Enveloppes Budgétaires dans les Paramètres, leur code apparaît en suggestion lors de la saisie d'une source.

Bilan financier global (vue 📊 Bilan)

La vue Bilan Financier Général consolide toutes les enveloppes budgétaires. Chaque source affiche sa capacité totale, le montant engagé par les projets en cours, et le reste disponible avec une jauge colorée (vert / orange / rouge).

Export financier

Depuis la vue Liste ou la vue Bilan, cliquez sur 💾 Excel pour exporter un tableau CSV compatible Excel avec tous les projets et leurs données budgétaires.

💾 Sauvegarde et restauration

Exporter une sauvegarde complète

1
Cliquez sur 💾 Export dans la barre de navigation.
2
Un fichier .json est téléchargé automatiquement. Il contient tous vos projets et paramètres.
3
Conservez ce fichier dans un dossier sécurisé (Drive, serveur…).

Restaurer une sauvegarde

1
Cliquez sur 📂 Charger dans la barre de navigation.
2
Sélectionnez votre fichier .json de sauvegarde.
3
L'application restaure toutes les données immédiatement.
⚠️ Restaurer un fichier de sauvegarde remplace toutes les données actuelles. Exportez d'abord si vous souhaitez conserver l'état courant.

Bonne pratique

💡 Exportez une sauvegarde après chaque session de travail importante. Nommez vos fichiers avec la date (ex : projets_2024-11-15.json) pour un archivage ordonné.

Exporter un projet individuel

Dans la liste des projets, cliquez sur l'icône 💾 d'un projet pour l'exporter individuellement en JSON. Ce fichier peut ensuite être partagé ou réimporté via 📥 Importer.

❓ Questions fréquentes

Mes données sont-elles sauvegardées automatiquement ? ▼
L'application enregistre vos données dans la mémoire locale de votre navigateur (localStorage). Elles persistent entre les sessions sur le même navigateur et le même ordinateur. Pour éviter toute perte (vidage du cache, changement d'ordinateur…), exportez régulièrement une sauvegarde JSON via le bouton 💾 Export.
Comment partager les données avec un collègue ? ▼
Exportez une sauvegarde complète (💾 Export) et envoyez le fichier JSON à votre collègue. Il pourra le charger via le bouton 📂 Charger. Il est aussi possible d'exporter un projet individuel (icône 💾 sur la ligne du projet) pour partager un seul dossier.
Comment imprimer une fiche de projet ou un document administratif ? ▼
Depuis la liste des projets, cliquez sur l'icône 🖨️ de la ligne souhaitée, ou ouvrez le projet et cliquez sur le bouton d'impression. Vous pouvez aussi aller dans la vue ✍️ Modèles pour choisir le type de document à générer. Utilisez ensuite Ctrl+P pour imprimer ou enregistrer en PDF.
Comment dupliquer un projet récurrent ? ▼
Dans le tableau de bord, cliquez sur l'icône 📋 (dupliquer) sur la ligne du projet à copier. Un nouveau projet identique est créé avec le statut "Brouillon". Il vous suffit de modifier les dates et les informations spécifiques à la nouvelle sortie.
Qu'est-ce que la Fiche Projet (EDD) ? ▼
La Fiche Projet (ou Fiche EDD) est un document JSON généré par une application complémentaire. Elle contient les informations pédagogiques approfondies : ODD, axes, dispositifs, partenaires, productions attendues. Importez-la via le bouton 🗂️ Fiche Projet pour l'associer automatiquement au projet correspondant.
Comment ajouter une nouvelle enveloppe budgétaire ? ▼
Allez dans ⚙ Paramètres → section Enveloppes Budgétaires. Cliquez sur + Ajouter et saisissez un code (ex : AP-01), un libellé et le montant disponible. Vous pouvez aussi importer une liste via CSV (téléchargez le modèle d'abord).
Le projet n'apparaît pas dans le Gantt, pourquoi ? ▼
Le Gantt affiche uniquement les projets avec une date de début renseignée. Vérifiez que le champ "Date Début" est bien rempli dans la section Logistique du formulaire.
Comment filtrer les projets par niveau ou statut ? ▼
Dans la vue 📋 Liste, utilisez la barre de recherche pour filtrer par texte, et la liste déroulante de niveau pour filtrer par classe. Pour filtrer par statut, saisissez le statut dans la barre de recherche (ex : "validé", "soumis").
J'ai effacé des données par erreur, comment les récupérer ? ▼
Si vous avez précédemment exporté une sauvegarde, rechargez-la via 📂 Charger. Si vous n'avez pas de sauvegarde, la récupération est malheureusement impossible. C'est pourquoi nous recommandons d'exporter régulièrement.
